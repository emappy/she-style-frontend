# She Style Frontend

Frontend application for the She Style E-Commerce platform.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Axios

## Features

- User authentication
- Product browsing
- Product search & filtering
- Shopping cart
- Checkout system
- Order history
- Admin dashboard
- Product management
- Order management
- Responsive design

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build Project

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=
```

## Deployment

Frontend deployed on Vercel.
